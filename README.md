# Radar Tracking Simulation

This project simulates a 2D aircraft motion at constant velocity. 
It is the foundation for adding radar measurement noise and filtering algorithms.

## Progress so far
- Created 2D motion model for aircraft
- Stored positions in arrays
- Verified code runs successfully

## Next Steps
- Add radar measurement noise
- Implement moving average / Kalman filter
- Plot true vs measured positions
- Analyse tracking accuracy
